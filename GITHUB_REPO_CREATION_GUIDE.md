# 🎯 GitHub仓库创建指南

## 步骤1：创建GitHub仓库

1. 打开浏览器，访问：https://github.com/new
2. 登录您的GitHub账户
3. 在"Repository name"字段输入：`word-pinyin`
4. 在"Description"字段输入：
   ```
   汉字拼音标注工具 - 支持DOCX和PDF导出，自动端口检测和浏览器打开功能
   ```
5. 选择仓库类型：
   - ✅ Public（公开）- 任何人都可以看到
   - ✅ Private（私有）- 只有您可以访问
6. **重要**：不要勾选"Initialize this repository with a README"
7. 不要添加`.gitignore`或license文件（我们已经有现成的）
8. 点击绿色的"Create repository"按钮

## 步骤2：获取仓库URL

创建完成后，您会看到类似这样的URL：
```
https://github.com/您的用户名/word-pinyin.git
```

例如，如果您的用户名是`johndoe`，那么URL就是：
```
https://github.com/johndoe/word-pinyin.git
```
