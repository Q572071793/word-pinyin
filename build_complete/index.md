# GitHub Pages 配置

这个文件用于配置GitHub Pages的部署设置。

## 部署说明

1. **源文件位置**: GitHub Pages将从 `build_complete/ui/` 目录提供静态文件
2. **主题**: 使用 jekyll-theme-minimal 主题
3. **基础URL**: `/word-pinyin`
4. **完整URL**: `https://q572071793.github.io/word-pinyin/`

## 访问地址

部署成功后，可以通过以下地址访问：
- 主页面: https://q572071793.github.io/word-pinyin/
- UI页面: https://q572071793.github.io/word-pinyin/ui/

## 功能

- 中文转拼音
- DOCX导出
- PDF导出
- 自动端口检测
- 浏览器自动打开
