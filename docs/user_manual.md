# 中文拼音生成器 - 用户操作手册

## 概述

中文拼音生成器是一个专业的工具，用于将中文文本转换为带有拼音标注的格式，并支持导出为DOCX文档。本工具采用表格布局，确保拼音和汉字的完美对齐，支持多种字体和格式选项。

## 主要功能

### 1. 文本转换
- 自动识别中文字符并添加拼音标注
- 支持长文本处理，自动分行显示
- 保留标点符号和特殊字符
- 智能处理多音字

### 2. 布局选项
- **表格布局**（推荐）：拼音在上，汉字在下，完美对齐
- **内联布局**：拼音显示在汉字上方，适合紧凑排版
- **括号布局**：拼音显示在汉字后面的括号内

### 3. 格式设置
- 每行字符数：可设置12-30个字符每行
- 拼音字体：Arial（英文字体，显示清晰）
- 汉字字体：Microsoft YaHei（微软雅黑，兼容性好）
- 边框显示：可选择是否显示表格边框

### 4. 文档导出
- 一键导出为DOCX格式
- 支持Microsoft Word、WPS Office等主流办公软件
- 文件包含完整的拼音和汉字对应关系

## 操作步骤

### 步骤1：输入文本
1. 在左侧文本框中输入需要转换的中文文本
2. 可以输入任意长度的中文内容
3. 支持复制粘贴功能

**示例文本**：
```
中文拼音生成器是一个专业的工具，用于将中文文本转换为带有拼音标注的格式。
```

### 步骤2：设置转换选项
1. **选择布局方式**：推荐使用"表格布局"
2. **设置每行字符数**：根据需求选择12-30个字符
3. **选择显示选项**：
   - 包含拼音：勾选此项以显示拼音
   - 显示边框：勾选此项以显示表格边框

### 步骤3：转换文本
1. 点击"转换拼音"按钮
2. 等待系统处理（通常1-3秒）
3. 在右侧预览区域查看转换结果

### 步骤4：预览和调整
1. 检查转换结果是否符合预期
2. 如需调整，可修改设置后重新转换
3. 确认无误后准备下载

### 步骤5：下载文档
1. 点击"下载DOCX"按钮
2. 选择保存位置
3. 文件将自动下载到指定位置

## 常见问题解决

### 问题1：下载的文件打不开
**解决方案**：
1. 确保使用Microsoft Word或WPS Office打开
2. 检查文件是否完整下载（文件大小应大于2KB）
3. 尝试重新下载

### 问题2：拼音显示异常
**解决方案**：
1. 确认系统已安装Arial和Microsoft YaHei字体
2. 检查文档编码设置
3. 尝试不同的布局方式

### 问题3：中文字符显示为方框
**解决方案**：
1. 安装中文字体包
2. 更新Office软件到最新版本
3. 检查系统语言设置

### 问题4：文件内容为空
**解决方案**：
1. 确认输入文本包含中文字符
2. 检查转换选项设置
3. 重新进行转换操作
