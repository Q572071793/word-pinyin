# 手动Git仓库设置指南

由于系统中未检测到Git命令行工具，这里提供手动设置Git仓库的步骤：

## 方法1：使用GitHub Desktop
1. 下载并安装GitHub Desktop：https://desktop.github.com/
2. 打开GitHub Desktop
3. 选择"Add Local Repository"
4. 选择项目目录：`D:\daima\word-pinyin`
5. 按照提示创建新的Git仓库
6. 添加远程仓库并推送

## 方法2：使用VS Code内置Git
1. 在VS Code中打开项目文件夹
2. 点击左侧的Git图标
3. 点击"Initialize Repository"
4. 添加所有文件并提交
5. 添加远程仓库并推送

## 方法3：手动创建Git结构
如果以上方法都不可用，可以手动创建Git仓库的基础结构。
